# Introdução ao Acapadev SDK

Bem-vindo à documentação detalhada do **Acapadev SDK**!

Este SDK serve como a ponte central de comunicação entre o Acapadev ID (a nossa base central de identidades) e a tua Aplicação Satélite.

## Porquê um SDK?
Ao integrar aplicações, escrever código de base repetitivo para validar assinaturas, capturar HTTP requests, descodificar JSON e gerir chaves pode ser entediante e arriscado (falhas de segurança). O SDK garante:
1. **Padronização:** Todas as aplicações da nossa rede falam a mesma língua.
2. **Segurança Máxima:** A encriptação e validação HMAC são geridas pelo pacote, sem que tenhas de mexer em código sensível.
3. **Agilidade:** Em menos de 2 minutos passas de uma app vazia para uma app completamente conectada ao Single Sign-On.
